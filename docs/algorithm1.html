Algorithm 1: Weighted k-Nearest Neighbors (k-NN)

This page summarizes the implementation of Algorithm 1 for approximate matching of multidimensional weighted data.

Overview

The Weighted k-NN algorithm ranks and returns the top k matches to a user's ideal preference by computing the weighted Euclidean distance between candidate items (e.g., homes) and the user's preference vector.

Features

  • Written in C with optimized data structures
  • Measures run-time performance for different input sizes
  • Matches 3-attribute candidate homes against weighted user preferences

How to Run

cd algo1_weighted_knn/src
gcc -o weighted_knn weighted_knn.c -lm
./weighted_knn
